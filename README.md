- styled-components

## **2. 실행 방법**

### version

- **node.js** v22.16.0  
  [설치하기 ->](https://nodejs.org/ko/download)
- **pnpm** v10.12.1  
  -> `npm i -g pnpm`

```
git clone frontend https://github.com/dolphin-kidnappers/AI-is-not-idiot.git
cd [clone project name]
pnpm i
pnpm run dev
```
